# Лабораторная работа №2

Тема: "Классы, конструкторы, деструкторы".

В проекте реализован вариант 4: класс `Box` с конструктором по умолчанию, параметризованным конструктором, конструктором копирования, деструктором и вычислением объема.

## Сборка

```powershell
cmake -S . -B build -G "Visual Studio 17 2022"
cmake --build build --config Release
```

## Запуск

```powershell
.\build\Release\lab2_box.exe
```
